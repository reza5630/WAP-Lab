Q2. What is the difference between a method and function?

A2. Function is a set of code to do some task in JavaScript. Method is also a function which is used as a property in Object.

Q3. What does 'this' refer to when used in a Java method?

A3. It might refer to the class itself, the methods of the class or the private instance variables of the class.

Q4. What does 'this' refer to when used in a JavaScript method?

A4. It refers to the "owner" of the method.

Q5. What does 'this' refer to when used in a JavaScript constructor function?

A5. The keyword this inside the constructor function points to the newly created object.

Q6. Assume object x is the prototype for object y in Javascript. Object x has a method f() containing keyword 'this'. When f is called by x.f(), what does 'this' refer to?

A6. It first refers to x(own), if property is not found it might refer to y(inherited) property.

Q7. What is a free variable in JavaScript?

A7. Free variables are simply the variables that are neither locally declared nor passed as parameter.

Q9. Write Javascript code for creating three Employee objects using the "new" keyword and a constructor function. Employee objects have the following fields: name, salary, position.

A9.
function Employee(name, salary, position) {
this.name=name;
this.salary=salary;
this.position=position;
}
var Wuletaw = new Employee("Wuletaw Adane", 120000, "Software Developer");

Q10. Write a Javascript function that takes any number of input arguments and returns the product of the arguments.

A10.
function product() {
let p=1;
for(let i=0; i<arguments.length; i++) {
p*=arguments[i];
}
return p;
}
